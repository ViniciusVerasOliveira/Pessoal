Falando um pouco sobre esse codigo: Aqui utilizei o codigo padrão gerado pela API do google maps como base e adaptei a necessidade da empresa, utilizando consulta ao Totvs(Protheus/ ADVPL), SQL(View/ Função). 
Você pode ver a pagina original neste link " https://galaxyled.com.br/NewSite22/onde-encontrar/ " 

Ao entrar na pagina é feito a pergunta para utilizar a localização atual, caso não aceite abre o mapa na posição do brasil, caso aceite ou preencha o endereço para pesquisar a pagina efetua uma consulta via Rest API e retorna os 5 clientes proximos.

Esta pagina foi um grande desafio pois quando comecei não conhecia está forma do JS, sabia o que era mas não entendia o funcionamento, ainda hoje confesso que não está perfeito mas para mim foi uma evolução que tive com esta pagina, onde passei de não saber ler a pagina e chegar a adapta-la de forma funcional.
